Frequently Asked Questions

Find answers to common questions about our plant disease detection system

PlantCare AI is an advanced plant disease detection system that uses artificial intelligence to identify diseases in plant leaves. Our system can detect 38 different plant diseases across various crops including apples, tomatoes, potatoes, grapes, and more.

Our AI model has been trained on a large dataset of plant images and achieves high accuracy in disease detection. The system provides confidence scores for each prediction, allowing users to understand the reliability of the results.

We support detection for various plants including:
  • Apple trees (scab, black rot, cedar apple rust)
  • Tomato plants (early blight, late blight, bacterial spot)
  • Potato plants (early blight, late blight)
  • Grape vines (black rot, leaf blight)
  • Corn (common rust, northern leaf blight)
  • And many more crops

Our system uses:
  • Machine Learning: TensorFlow/Keras with Convolutional Neural Networks
  • Web Framework: Flask (Python)
  • Frontend: HTML5, CSS3, Bootstrap 5, JavaScript
  • Image Processing: PIL/Pillow for image preprocessing

The AI model uses a Convolutional Neural Network (CNN) that has been trained on thousands of plant images. When you upload an image, it:
  1. Preprocesses the image to 128x128 pixels
  2. Analyzes the visual features using the trained model
  3. Compares against known disease patterns
  4. Provides a prediction with confidence score

Yes, we take data security seriously. Uploaded images are processed securely and are not stored permanently. We use secure protocols and follow best practices for data protection.

Using our disease detection is simple:
  1. Go to the "Detect" page
  2. Upload a clear image of the plant leaf
  3. Click "Analyze Image"
  4. View the results and treatment recommendations

We support common image formats including:
  • JPEG (.jpg, .jpeg)
  • PNG (.png)
  • GIF (.gif)
  • BMP (.bmp)
  • TIFF (.tiff)

For best results:
  • Use clear, well-lit images
  • Focus on the affected area
  • Include the entire leaf if possible
  • Avoid shadows and reflections
  • Use high-resolution images
  • Take photos from multiple angles
